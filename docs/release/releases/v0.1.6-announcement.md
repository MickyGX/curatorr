# Curatorr v0.1.6 Announcement

Curatorr `v0.1.6` is out.

## Highlights

- Plex and Tautulli webhooks now use a shared-secret protected Curatorr URL instead of accepting unauthenticated webhook traffic.
- The Tautulli Settings button now creates or updates the Curatorr webhook with the correct URL, triggers, and JSON body.
- Service credentials are better protected in Settings, and credentialed Plex/Tautulli calls no longer rely on query-string authentication.
