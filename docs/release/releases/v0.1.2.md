# Curatorr v0.1.2

## Summary

- Fixed the first-run setup flow so creating the setup admin no longer drops users into a duplicate second admin-account screen.
- Fixed wizard routing so users who have not completed onboarding are sent into the personal wizard consistently, including Plex SSO users.
- Fixed the personal wizard to build the master track cache before rendering when needed, so the favourite-genres step has actual options on fresh installs.
- Tightened local development ignores for temp compose and temp data folders used during testing.

## Upgrade Notes

- No migration steps are required.
- On a brand-new install, the first visit to the favourite-genres step may take a little longer while Curatorr builds the initial master track cache.
