# Curatorr v0.1.4

## Summary

- Fixed onboarding redirects so locally created Curatorr accounts are not forced into the server wizard after account creation or login.
- Plex accounts still auto-run onboarding when setup is incomplete, preserving the intended guided flow for external users.

## Upgrade Notes

- No migration steps are required.
- Local Curatorr users can still open `/wizard` or `/wizard/user` manually when needed.
