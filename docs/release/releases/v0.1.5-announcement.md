# Curatorr v0.1.5 Announcement

Curatorr `v0.1.5` is out.

## Highlights

- The Plex token now stays visible in the masked settings field after save, instead of looking like it was cleared.
- Stored-secret state is now rendered consistently across Plex, Tautulli, and Lidarr settings.
- The About page now shows recent releases and real changelog highlights, matching the Launcharr-style release modal.
