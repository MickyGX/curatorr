# Curatorr v0.1.8 Announcement

Curatorr `v0.1.8` is out.

## Highlights

- Long Curatorr pages embedded in Launcharr can now scroll fully to the bottom.
- Launcharr embed mode now uses the correct page-height and overflow behavior without affecting standalone Curatorr.
