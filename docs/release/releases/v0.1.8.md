# Curatorr v0.1.8

## Summary

- Fixed Launcharr embed scrolling so long Curatorr pages can reach the true bottom of the page instead of being clipped inside the embedded shell.
- Corrected the embedded layout sizing rules used by Launcharr mode, allowing document-level scrolling without changing the normal standalone Curatorr layout.

## Upgrade Notes

- Rebuild and redeploy Curatorr anywhere it is embedded in Launcharr so the updated Launcharr-specific layout CSS is included.
- If Curatorr is already open inside Launcharr after upgrade, do a hard refresh of the Launcharr page to reload the iframe styles.
