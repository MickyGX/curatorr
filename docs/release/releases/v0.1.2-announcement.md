# Curatorr v0.1.2 Announcement

Curatorr `v0.1.2` is out.

## Highlights

- The setup flow no longer repeats admin account creation.
- Local and Plex logins now enter the personal wizard properly when onboarding is incomplete.
- Fresh installs now populate the favourite-genres step reliably by warming the music cache before rendering it.
