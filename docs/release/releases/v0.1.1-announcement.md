# Curatorr v0.1.1 Announcement

Curatorr `v0.1.1` is out.

## Highlights

- Launcharr embed mode now lets Curatorr behave like a native dashboard app inside an iframe.
- Embedded mode now follows Launcharr theme and corner styling more cleanly, with support for explicit embed origin allow-listing.
- Tautulli setup is easier to recover from, with a manual webhook registration action available in Settings.
