# Curatorr v0.1.1

## Summary

- Added Launcharr embed mode support so Curatorr can render cleanly inside an iframe without its own sidebar, title bar, or outer chrome.
- Added theme-aware embedded rendering, including Launcharr light/dark sync and square-corner compatibility.
- Added configurable cross-origin embedding through `EMBED_ALLOWED_ORIGINS`.
- Added a manual Tautulli webhook registration action in Settings for installs that need to retry or complete setup after the wizard.
- Improved Tautulli setup behavior by falling back to Curatorr's saved local or remote URL when registering the webhook.
- Expanded docs for embed mode, configuration, integrations, and artist/Lidarr behavior.

## Upgrade Notes

- If you embed Curatorr in Launcharr or another dashboard on a different origin, add that dashboard origin to `EMBED_ALLOWED_ORIGINS`.
- If your existing install skipped Tautulli webhook auto-registration, you can now register it from Settings without rerunning the setup wizard.
