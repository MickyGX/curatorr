# Curatorr v0.1.5

## Summary

- Fixed Plex settings so the saved Plex token stays in the masked input after save instead of appearing to vanish.
- Fixed the settings view state for stored secrets, so Plex, Tautulli, and Lidarr settings can consistently reflect whether credentials are already saved.
- Added Launcharr-style release loading in About settings, including a populated Releases list and changelog modal highlights sourced from local release note files.

## Upgrade Notes

- No migration steps are required.
- Rebuild the container so the updated settings and release-note views are included in the running image.
