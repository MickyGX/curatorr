# Curatorr v0.1.7 Announcement

Curatorr `v0.1.7` is out.

## Highlights

- Plex is now the preferred live playback source, with a settings toggle to compare Plex and Tautulli webhook ingestion when needed.
- Playback tracking now follows the full session lifecycle, which improves skip detection, belter/decent scoring, and duplicate prevention for paused, resumed, or mixed playback.
- Tautulli has been repositioned as a cleaner backup path, while Lidarr recommendation handling now exposes manual vs automatic actions more clearly and applies Curatorr tags to adds.
